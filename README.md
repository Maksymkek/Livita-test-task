# Livita Test Task

## Requirements

- **Xcode 26**
- **iOS 26 Simulator**

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Maksymkek/Livita-test-task.git
```

2. Open the project in **Xcode 26**.

3. Select an **iOS 26 Simulator** and run the app.

## Tested Devices

The layout was tested on the following devices:

- **iPhone 17 Pro**
- **iPhone 17 Pro Max**
